* Basic Keybinds
 [U] Update Elfeed
 [s] Search
 [E] Edit
 [g] Refresh counts
 [q] Kill dashboard

* Feeds
** [u] Unread ([[elfeed:+unread][258]])
 [n] New      ([[elfeed:+unread @1-week-old][ 0]])

 [b] Blogs    ([[elfeed:+unread +blog][ 0]])
 [y] Youtube  ([[elfeed:+unread +youtube][ 0]])
 [N] News     ([[elfeed:+unread +news][ 0]])
